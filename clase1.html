Anatomia de una declaracion

Es un declaracion

p {
color:red
}

p: selector

color: propiedad

red: propiedad

Tipos de selectores

Univelsal:

* {
margin 0;
}

Tipo:

h1 {
color:red;
}

Clase:

.saludo {
font-size:2em;
}

Id:

#id {
borde-radio: 20px;
}

Pseudo-clases

Primer hijo

p:first-child {
color:white;
}

Ultimo hijo

p:last-child{
color: purple;
}

hijos pares

p:ntn-child(2n){
color_red;
}

hijos inpares

p:ntn-child(2n+1){
color_blue;
}

Documentacion

Pseudo-elementos

Primera letra

p::first-letter {
color:red;
font-size: 20px;
}
